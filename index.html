Description of the Membranes Problem  我们假设弹性薄膜 \(\Omega\) 被固定在硬性支撑 \(\Gamma\) 上，同时在表面的每个单元 \(\mathrm{d}x=\mathrm{d}x_1\mathrm{d}x_2\) 上均施加有大小为 \(f(x)\mathrm{d}x\) 的外力。薄膜在垂直方向上的位移 \(\phi(x)\) 可以通过求解Laplace方程来得到 \[\begin{aligned}-\Delta \phi=f\ in\ \Omega\end{aligned}\]  如果固定薄膜的支撑是一个平面,那么薄膜的垂直位移满足:\[\phi\big|_\Gamma=0\]  但是如果固定薄膜的支撑并不是一个平面,而是在点 \((x_1,x_2)\) 处的高度为 \(z(x_1,x_2)\) ,那么其满足非齐次Dirichlet边界条件. \[\phi\big|_\Gamma=z\]  如果薄膜边界的一部分 \(\Gamma_2\) 不固定在支撑上,而是允许自由滑动,由于薄膜本身刚性性质,\(\Gamma_2\) 上的任意一点的法向量 \(\vec{n}\) 是0,因此边界条件为 \[\phi\big|_{\Gamma_1}=z,\frac{\partial \phi}{\partial \vec{n}}\big|_{\Gamma_2}=0\] 其中 \(\Gamma_1=\Gamma-\Gamma_2\) Examples  如果有一个椭圆，其半长轴的长度为 \(a=2\)，半短轴的长度为 \(b=1\)，在椭圆表面施加一个大小为1的外力，利用Freefem++编程如下：  网格剖分结果图  解的结果图  buildmesh表示建立区域的三角剖分,它调用了一个基于的Delaunay测试的三角剖分子程序.Delaunay测试首先通过三角化边界点再通过细分边加入内部点.通过其得到的三角剖分的精细程度由最近的一条边界的尺寸来确定.  为了验证这个方法的有效性,我们通过给出一个特殊的参数,来精确求解误差及收敛率.在单位圆上,当 \(z=0,f=-4(\cos(x^2+y^2-1)-(x^2+y^2)\sin(x^2+y^2-1))\),其精确解为\(\phi_e=\sin(x^2+y^2-1)\),但其在\(\Gamma_2\)上法向量并非为0,而是2.利用Freefem++编程如下  结果如下所示

Thm(Wilf,1967) Let G be a graph. Then \(\chi(G)\leq\lambda_1(G)+1\), where \(\chi(G)\) is the chromatic number

A proper vertex coloring is a map \(c:V(G)\to\mathbb{N}\) s.t. \(c(v_i)\not=c(v_j)\) if \(v_iv_j\in E(G)\).

A graph is a k-colorable if there exists a proper vertex coloring \[ c:V(G)\to\{1,2,\cdots,k\}=[k]. \]

Thm(Eigenvalue Interlacing Theorem)
Let A be a symmetric real \(n\times n\) and let B be an m-th principal submatrix(obtained by deleting both i-th row and i-th column for some n-m values of i). Suppose A has eigenvalues \(\lambda_1\geq\cdots\geq\lambda_n\), and B has eigenvalues \(\beta_1\geq\cdots\geq\beta_m\).Then \[ \lambda_k\geq\beta_k\geq\lambda_{k+(n-m)},\qquad k=1,2,\dots,m \]
